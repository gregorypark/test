# Data

## Source

Download the "Most Recent Institution-Level Data" from:
https://collegescorecard.ed.gov/data/

## Specific Files

1. `Most-Recent-Cohorts-Institution.csv` - Dataset
2. `CollegeScorecardDataDictionary.xlsx` - Variable definitions

## Instructions

1. Visit https://collegescorecard.ed.gov/data/
2. For the dataset, click "Download The Data" → "Most Recent Institution-Level Data"
3. For the data dictionary, click "Data Documentation" → "Data Dictionary"

** The CSV file is too large and not included in this repository.

## Key Variables Used

- `INSTNM` - Institution name
- `STABBR` - State where university is located
- `MD_EARN_WNE_P10` - Median earnings 10 years after entry
- `PCIP##` - Percentage of degrees in each major
- `PREDDEG` - Predominant degree type
